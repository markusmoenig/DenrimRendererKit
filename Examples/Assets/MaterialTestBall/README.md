# Material Test Ball Assets

These meshes come from Benedikt Bitterli's Material Test Ball scene and are covered by the CC0/public-domain license in `LICENSE.txt`.

Only the mesh geometry is copied here. The original scene's light probe is intentionally not included because its source license is non-commercial.

`Meshes/FrontFacing` contains Denrim-ready baked copies of the three source PLY parts. They apply the original PBRT instance scale/translation plus a shared `-0.65` radian Y rotation around the preview object center, so the test ball faces a fixed front camera while the room remains axis-aligned.
